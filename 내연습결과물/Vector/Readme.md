## `Vector` 내용 정리

`vector`는 `List`를 뜻하는 것이 아니라, 차원을 표현하기 위한 자료구조. 
  
C# `vector` O  
C++ `vector` X  

1. `vector`의 절대 크기를 뜻하는 magnitude  
2. `vector`의 크기를 1로 만드는 정규화 normalize
3. 더하기 add, 빼기 sub, 곱하기 mul, 나누기 div ... 앞에 서술된 사칙연산은 `static`함수를 쓸 수도있다. 

수학적인 내용들은 너무 기본적인 것이라 넘어간다.  
하지만 `p5.js`와 친해지기 좋은 내용이여서 충분히 처음부터 끝까지 따라해볼만 하다. 또한 p5.js로   
스케치를 효과적으로 하기위해서 객체지향적인 설계를 하는데에 기초를 닦기도 좋다.   
마지막으로, 예제가 간단하기에 조금 조금씩 조건을 변경해주면 바뀌는 반응에 따라 논리적으로 추론하는 연습도 또한 하기 좋다.
